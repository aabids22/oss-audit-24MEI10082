# OSS Capstone Project

This project is based on understanding and using open source software in a practical way.The selected software for this project is VLC Media Player.

## Project Overview

The project includes installation of VLC on Ubuntu system,usage of basic Linux commands,understanding file permissions and user access.

It also includes ecosystem study of VLC,comparison between open source and proprietary software,and shell scripting tasks.

## Learning Outcome

Through this project,I was able to understand how open source software works in real environment and how Linux commands and scripting can be used.

In many parts,it was simple,but some parts required careful typing and understanding,especially scripting part.

Overall,this project helped me to learn both theoretical and practical aspects of open source software.
